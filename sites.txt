# PriceAnalyzer – Site Yapılandırma Dosyası
#
# Format (pipe | ile ayrılmış 4 alan):
#   SiteAdı | AramaURLŞablonu | FiyatCSSSeçici | StokCSSSeçici
#
# Kurallar:
#   - {code} yer tutucusu ürün koduyla değiştirilir
#   - # ile başlayan satırlar yorum satırıdır
#   - Boş satırlar atlanır
#   - CSS seçiciler, BeautifulSoup'un select_one() metoduyla kullanılır
#
# ──────────────────────────────────────────────────────────────────
# Örnek Yapılandırmalar (Kendi sitelerinize göre düzenleyin)
# ──────────────────────────────────────────────────────────────────

# Tedarikci1 | https://www.tedarikci1.com/ara?q={code} | .price | .stock-status

# Tedarikci2 | https://www.tedarikci2.com/product/{code} | #product-price | #availability
